```shell
# if you don't enable go module:  ┻━┻︵╰(‵□′)╯︵┻━
```

```shell
# go build
go build -o ${the_name_you_want} ./icleaner.go
```

#### usage

```shell
# use -h to show usage
./${the_name_you_want} -h
```

### For Direct Usage

```shell
# use -h to show usage
```
